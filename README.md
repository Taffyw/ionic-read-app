Ionic Read App
=====================

一个简单的阅读类app实现了 主要实现了用户登录注册 收藏书籍 查看书籍 评价 阅读等功能

## 使用这个项目

安装最新版本的 Ionic CLI:
...src\providers\Constants.ts 修改项目api地址为本地后台服务地址 
在这之前需要运行express后台服务 [express-read-app](https://github.com/Taffyw/express-read-app)

```bash
$ npm install -g ionic
```
Then run:

```bash
$ ionic serve
```
```bash
$ ionic platform add android
$ ionic platform add ios
```
```bash
$ ionic run android
$ ionic run ios
```

More info on this can be found on the Ionic [Getting Started](http://ionicframework.com/docs/v2/getting-started/) page.
